Ce projet est un Système Médical Distribué conçu pour assurer le suivi des patients en temps réel. Il repose sur une architecture hybride alliant Java RMI pour les interactions synchrones (consultation des dossiers) et JMS (via Apache ActiveMQ) pour la communication asynchrone. Cette conception permet de découpler l'interface utilisateur du traitement des données : les alertes vitales critiques sont transmises instantanément aux médecins via une file prioritaire, tandis que l'archivage en base de données se fait en parallèle, garantissant fluidité et fiabilité.
